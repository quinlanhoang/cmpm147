Experiment 2 - Living Impressions

Description

The objective is to create a live impression of some landscaped image using procedural generation. This is to mimic the general scope of the image with relativve colors, primitives, and polygons:

Technical

Relation to inspring image:

- Color: The sky, sand, trees, bushes, grass and rock colors were sampled from the inspiring image using the Photoshop eyedropper tool.

- Primitives: The trees are rendered with vertex adjustments and the bushes are rendered with ellipses.

- Polygons: The background mountain is a single randomly generated polygon using `vertex` calls as well as the grass.

- Polygons: The background mountain is a single randomly generated polygon using `vertex` calls as well as the grass.

- Life: The position of trees is animated over time as if to simulate the view from a passing train and the slightly responds to the horizontal motion of the mouse cursor.

Reflection

Overall, the assignment was pretty doable. There was some difficulty with finding the right metrics to reciprocate similar images in terms of the mountain range or the trees as you would have to reguarly adjust vertices or steps in order to recreate a similar given shape.
